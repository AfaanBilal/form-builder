AMX Form Builder

AMX Form Builder is a web app that helps users create simple HTML forms with PHP backend using drag-n-drop. The generated PHP files may be uploaded on any PHP-enabled web server.

Getting Results

The submitted data will be stored in a CSV file in the same directory as the php file with the same name as that of the form. The file may be opened in MS Excel easily or any text-editor.
